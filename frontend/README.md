# 🐾 Virtual Pet Manager

Aplicación Full Stack moderna para gestionar mascotas virtuales.

## 🚀 Stack Tecnológico

| Frontend | Backend | Base de Datos |
|----------|---------|---------------|
| React + Vite | Node.js + Express | MongoDB + Mongoose |
| Bootstrap 5 | MVC Architecture | Local Instance |
| Axios | REST API | CRUD Completo |
| React Router | Middleware | Validaciones |

## 📋 Funcionalidades

- ✅ CRUD completo de mascotas
- ✅ Filtros por especie y estado de ánimo
- ✅ Búsqueda en tiempo real
- ✅ Dashboard con estadísticas
- ✅ Diseño responsive
- ✅ Validaciones frontend y backend

## 🛠️ Instalación

```bash
# Backend
cd backend && npm install && npm run dev

# Frontend (nueva terminal)
cd frontend && npm install && npm run dev
```

## 🌐 API Endpoints

| Método | Ruta | Descripción |
|--------|------|-------------|
| GET | /api/pets | Obtener todas |
| GET | /api/pets/:id | Obtener una |
| POST | /api/pets | Crear |
| PUT | /api/pets/:id | Actualizar |
| DELETE | /api/pets/:id | Eliminar |
